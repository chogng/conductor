# Conductor Studio User Guide

## Import Data

- Open a folder or import supported table files from the Files area.
- After import, check the file preview, detected type, and template match before processing.

## Templates and Slicing

- Use Set with Template on a file item to confirm columns, variables, and legend labels.
- Use Slice with Template when one source file needs to be split into template-sized segments.

## Charts and Search

- Review curves, legends, scales, and axis settings in the Chart area.
- Use Search to locate curve points by X value when comparing files.

## Export

- Generate Origin outputs from the Export area after the curves and chart settings are confirmed.

## Notes

This guide is bundled with the app and rendered through the same safe Markdown pipeline as release notes.
