# Conductor Studio 使用说明

## 导入数据

- 在文件区域打开文件夹或导入支持的表格文件。
- 导入后先确认文件预览、识别类型和模板匹配是否符合预期。

## 模板与切片

- 在文件项菜单中选择“设置模板”来确认数据列、变量和图例。
- 需要把一个源文件拆分为模板长度的片段时，使用“以模板切片”。

## 图表与搜索

- 在图表区域检查曲线、图例、缩放方式和坐标轴设置。
- 对比文件时，可以使用搜索按 X 值定位曲线点。

## 导出

- 确认曲线和图表设置后，在导出区域生成 Origin 输出。
