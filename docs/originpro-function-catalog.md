# originpro 接口功能分组手册

本手册按“功能场景”整理 `originpro` 的常用可调参数，主要回答四个问题：曲线样式怎么调、坐标轴怎么调、图例怎么调、如何映射到本项目 `capabilities` 配置。

## 1. 说明

| 项目 | 说明 |
| --- | --- |
| 运行方式 | External Python 和 Embedded Python 使用同一套 `originpro` API |
| 文档基线 | 以官方 `originpro` 参考页（`Generated by 1.12.0`）和官方示例页为准 |
| 高级兜底 | 样式项无封装接口时，优先用 `op.lt_exec(...)` 或 `plot.set_cmd(...)` |

## 2. 功能总览

| 功能 | 主要接口 | 可调重点 |
| --- | --- | --- |
| 会话控制 | `op.set_show`, `op.lt_exec`, `op.detach` | Origin 可见性、执行 LabTalk |
| 数据导入 | `DSheet.from_file`, `Connector` | 连接器类型、选择器、导入行为 |
| 建图与图层 | `GLayer` | 加图、分组、缩放、范围、布局 |
| 曲线样式 | `Plot` | 颜色、符号、透明度、填充、命令透传 |
| 坐标轴 | `Axis` | 标题、scale、limits、步长 |
| 图例与标注 | `GLayer.label`, `Label` | 文本、边框、位置、对象属性 |
| 导出 | `GPage.save_fig` | 导出格式、覆盖、尺寸 |
| 兜底高级项 | LabTalk | API 未封装项 |

## 3. 数据导入

### 3.1 `DSheet.from_file(...)`

```python
from_file(fname, keep_DC=True, dctype='', sel='', sparks=False)
```

| 参数 | 含义 |
| --- | --- |
| `fname` | 文件路径 |
| `keep_DC` | 是否保留 Data Connector |
| `dctype` | 连接器类型，如 `HTML` / `MATLAB` / `NetCDF` / `Import Filter` |
| `sel` | 连接器选择器，如 `Tables/_6` |
| `sparks` | 是否创建 sparklines |

### 3.2 `Connector`

常用在复杂导入流程中：

```python
conn = wks.from_file(path, dctype='CSV')
# conn.settings(...)
# conn.imp(...)
```

可调重点：

- 导入前设置（列类型、表选择、过滤规则）
- 导入执行方式（是否重导入、保留连接）

## 4. 建图与图层（GLayer）

### 4.1 新增曲线与分组

```python
gl.add_plot(obj, coly=-1, colx=-1, colz=-1, type='?', colyerr=-1, colxerr=-1)
gl.group(group=True, begin=-1, end=-1)
```

| 参数 | 含义 |
| --- | --- |
| `obj` | 数据对象（工作表/矩阵等） |
| `coly` / `colx` / `colz` | 作图列索引 |
| `type` | 作图类型（如 `'l'`, `'s'`, `'y'`, `'c'`）或类型编号 |
| `colyerr` / `colxerr` | 误差列 |
| `group` | 是否分组 |
| `begin` / `end` | 指定分组范围 |

### 4.2 范围与重缩放

```python
gl.rescale(skip='')
gl.set_xlim(begin=None, end=None, step=None)
gl.set_ylim(begin=None, end=None, step=None)
gl.set_zlim(begin=None, end=None, step=None)
```

| 参数 | 含义 |
| --- | --- |
| `skip` | 重缩放时跳过哪些轴，可组合 `x/y/z/m` |
| `begin` / `end` | 轴范围 |
| `step` | 主刻度步长 |

### 4.3 布局与标注

```python
gl.arrange(nr, nc)
gl.label(text, x=0, y=0, color=None)
gl.remove_label(text)
```

| 参数 | 含义 |
| --- | --- |
| `nr` / `nc` | 图层排布（行/列） |
| `text` | 标签文本（图例也可按标签对象处理） |
| `x` / `y` | 标签位置 |
| `color` | 标签颜色 |

## 5. 曲线样式（Plot）

### 5.1 主要属性

| 属性 | 含义 |
| --- | --- |
| `plot.color` | 曲线颜色（`int` / 颜色名 / RGB tuple） |
| `plot.colormap` | 调色板 |
| `plot.colorinc` | 组图颜色递增（bool） |
| `plot.symbol_kind` | 符号类型（int） |
| `plot.symbol_kindinc` | 组图符号递增（bool） |
| `plot.symbol_size` | 符号大小（float） |
| `plot.symbol_sizefactor` | 组图符号尺寸比例（float） |
| `plot.symbol_interior` | 符号填充模式（int） |
| `plot.transparency` | 透明度（0-100） |

### 5.2 主要方法

```python
plot.set_shapelist(name)
plot.set_fill_area(above=-1, type=9, below=-1)
plot.set_cmd(*args)
```

| 方法 | 可调参数 |
| --- | --- |
| `set_shapelist(name)` | 符号列表/主题名 |
| `set_fill_area(...)` | `above`, `type`, `below` |
| `set_cmd(*args)` | 直接传 `set` 命令参数，适合高级样式 |

## 6. 坐标轴（Axis）

### 6.1 核心接口

```python
ax.title = 'VGS (V)'
ax.scale = 'linear'
ax.set_limits(begin=None, end=None, step=None)
```

| 接口 | 可调参数 |
| --- | --- |
| `title` | 标题文本 |
| `scale` | 轴类型（字符串或枚举整数） |
| `set_limits(...)` | `begin`, `end`, `step` |
| `sfrom`, `sto`, `sstep` | 起点、终点、步长（直接属性） |

### 6.2 常用 `scale` 字符串

- `linear`
- `log10`
- `probability`
- `probit`
- `reciprocal`
- `offset_reciprocal`
- `logit`
- `ln`
- `log2`

## 7. 图例与标注（Legend / Label）

官方示例常见写法：

```python
gl = gp[0]
lgnd = gl.label('Legend')
lgnd.text = '\\l(1) Id-Vg'
lgnd.set_int('showframe', 0)
```

可调项：

| 接口 | 用途 |
| --- | --- |
| `label.text` | 图例/标签文本 |
| `label.set_int(...)` | 整型属性（如 `showframe`） |
| `label.set_float(...)` | 浮点属性（位置/尺寸相关） |
| `label.set_str(...)` | 字符串属性 |
| `label.remove()` | 删除对象 |

## 8. 导出

```python
gp.save_fig(path, type='auto', replace=True, width=0)
```

| 参数 | 含义 |
| --- | --- |
| `path` | 导出路径 |
| `type` | 导出类型（自动或指定） |
| `replace` | 是否覆盖已有文件 |
| `width` | 导出宽度 |

## 9. 兜底：LabTalk 参数入口

`originpro` 未封装时，优先用：

- `op.lt_exec("...;")`
- `plot.set_cmd(...)`

这样仍然走同一条 `originpro` 自动化链路。

## 10. 本项目可直接配置的参数

### 10.1 Worker CLI 参数

- `--plot-type`
- `--xy-pairs`
- `--plot-command`
- `--post-plot-command`（可重复）

### 10.2 `capabilities` JSON 分区

- `import`: `workbookLongName`, `preCommands`, `postCommands`
- `plot`: `command`, `preCommands`, `postCommands`
- `graph`: `preCommands`, `postCommands`
- `style`: `commands`
- `axis`: `commands`
- `commands`: `preCommands`, `postCommands`

### 10.3 执行顺序

1. `commands.preCommands`
2. `import.preCommands` -> 导入 -> `import.postCommands`
3. `graph.preCommands`
4. `plot.preCommands` -> Plot 命令 -> `plot.postCommands`
5. `style.commands`
6. `axis.commands`
7. `graph.postCommands`
8. `commands.postCommands`

### 10.4 `capabilities` 示例

```json
{
  "import": {
    "workbookLongName": "IdVg_2026Q1",
    "preCommands": ["newbook;"],
    "postCommands": ["wks.nCols=6;"]
  },
  "plot": {
    "command": "plotxy iy:=((1,2),(3,4)) plot:=202;",
    "postCommands": ["legend -r;"]
  },
  "style": {
    "commands": [
      "set %c -c color(blue);"
    ]
  },
  "axis": {
    "commands": [
      "layer.x.from=-2; layer.x.to=2;",
      "layer.y.type=2;"
    ]
  }
}
```

## 11. 最小可用示例

```python
import originpro as op

op.set_show(True)
op.lt_exec("newbook;")
op.lt_exec('impCSV fname:="C:\\\\data\\\\demo.csv";')
op.lt_exec("plotxy iy:=((1,2)) plot:=202;")

gp = op.find_graph()
gl = gp[0]
p = gl.plot_list()[0]

p.color = "Blue"
p.symbol_kind = 2
p.symbol_size = 4.5
p.transparency = 15

x = gl.axis("x")
y = gl.axis("y")
x.title = "VGS (V)"
y.title = "ID (A)"
y.scale = "log10"
y.set_limits(1e-12, 1e-3, None)

lgnd = gl.label("Legend")
lgnd.text = "\\l(1) Demo"
lgnd.set_int("showframe", 0)

gl.rescale()
```
