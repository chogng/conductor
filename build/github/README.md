# GitHub Build Scripts

This directory holds helper scripts that are shared by GitHub Actions release workflows.

## Naming

- `verify-*`: configuration and release checks.
- `upload-*`: release asset upload helpers.
- `test-*`: lightweight script checks.

## Current Use

- `verify-release-tag.ps1`
- `verify-source-repository.ps1`
- `upload-release-assets.ps1`
